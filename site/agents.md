# Instructions

## English Page
- The English resume is stored in `resume_en.md`.
- The site serves language versions from `index.html` using a URL query parameter:
  - Japanese: `./?lang=ja` (loads `resume.md`)
  - English: `./?lang=en` (loads `resume_en.md`)
- Keep `resume.md` and `resume_en.md` structurally aligned (same section order and page-break markers) so styling and PDF export behavior stay consistent.
